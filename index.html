Sophisticated Cryptography for Absolute Privacy

Designed from the ground up with no central tracking servers, sign-ups, logs, or stored credentials.

vpn_key

End-to-End Encryption

All messages are securely encrypted with high-entropy **AES-GCM** keys derived on-the-fly and signed locally with the sender's private key before hitting the blockchain ledger.

lock_clock

Active Autolock (RAM Sanitization)

The Chromium MV3 alarm architecture enforces a custom idle autolock threshold. Private keys and secure payloads are securely purged from your RAM space when idle to prevent extraction.

cloud_queue

Encrypted IPFS Storage

Send rich media, files, or documents securely. Raw items are automatically encrypted in-browser before being securely deployed to decentralized IPFS gateways via Pinata API routines.

Decentralized Transaction Architecture

AlloChat stores no conversation logs or profiles. Your chats exist solely as encrypted, secure data points inside Algorand transaction note spaces.

01

Keys Generation

Your cryptographic keys and Algorand address are derived entirely locally from a secure BIP-39 25-word mnemonic seed. Secret credentials never escape the memory context of your local browser.

02

Encryption & Broadcast

Before dispatching, your text is securely encrypted using Diffie-Hellman (X25519) key exchange setups. It is then published as a 0 ALGO transaction to the blockchain. The flat cost of sending is just 0.001 ALGO per write.

03

Asynchronous Background Syncer

A background service worker, highly optimized for Manifest V3 extension standards, polls Algonode API block indexes for incoming cryptographic notes matching your address hash, instant-notifying you without battery drain.

Direct Extension Installation (Load Unpacked)

As an audited, fully open-source security tool, you install AlloChat as an unpacked directory manually, isolating it from third-party runtime modifications.

1
Download the Repository Packages

Click here to download the repository as a ZIP archive or clone it using git with git clone https://github.com/devitamine/AlloChat.git. Once downloaded, extract the contents and locate the main production extension bundle folder: /allochat-extension-mainnet/.

2
Open Extensions Panel in Chromium

In your address bar (supports Chrome, Brave, Arc, Edge, Opera, Vivaldi), type: chrome://extensions/ and press Enter.

3
Activate Developer Mode

In the top right-hand corner of the browser extensions manager page, toggle on the **Developer mode** slider.

4
Load the Unpacked Directory

Click the utility button **Load unpacked** in the top-left area. Select the local copy of your /allochat-extension-mainnet/ directory. Pin the icon, and secure your chats!
